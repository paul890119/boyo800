## 🎮 遊戲模式

### 1️⃣ 認讀挑戰 (Recognition)
**難度：⭐ 簡單**

看到中文提示，選擇正確的英文單字。動畫風格的 UFO 防禦戰鬥。

```
例子：
中文提示 → "媽媽"
選項：mother, sister, father, brother
→ 點擊 mother 消滅 UFO
```

**適合：** 初學者、快速複習

---

### 2️⃣ 字母重組 (Scramble)
**難度：⭐⭐ 中等**

打散的字母需要重新排列成正確的單字。鍛鍊拼寫能力。

```
例子：
中文提示 → "貓"
打散字母 → c, a, t
操作 → 按順序點擊：c → a → t
確認 → cat
```

**適合：** 練習拼寫、強化記憶

---

### 3️⃣ 拼字測驗 (Spelling)
**難度：⭐⭐⭐ 困難**

聽到英文發音，在輸入框拼出正確的單字。最有效的學習模式。

```
例子：
中文提示 → "狗"
聽發音 → /dɔg/
輸入 → dog
確認 → Enter
```

**適合：** 提升實力、深度學習

## ⚙️ 難度設置

### 難度參數對比

| 參數 | 簡單 | 普通 | 困難 |
|------|------|------|------|
| ⏱️ 每題時間 | 150ms | 100ms | 60ms |
| ❤️ 額外生命 | +2 | 0 | -1 |
| 💰 分數倍增 | 1x | 1.5x | 2.5x |
| 推薦人群 | 初心者 | 一般玩家 | 高手 |

### 分數計算

```
基礎分 = 10 + 時間獎勵 + 連擊獎勵
時間獎勵 = 剩餘時間 / 10
連擊獎勵 = 當前連擊數 × 5
最終分 = 基礎分 × 難度倍增
```

## ❓ 常見問題

### Q1: 為什麼音效不播放？
**A:** 
- 檢查瀏覽器音量和系統音量
- 某些瀏覽器需要用戶交互後才能播放音效
- 嘗試在遊戲中禁用音效後重新啟用

### Q2: 進度如何保存？
**A:** 進度自動保存到瀏覽器 LocalStorage，關閉瀏覽器後再次打開仍可恢復進度。

### Q3: 可以在不同設備上同步進度嗎？
**A:** 目前暫不支持，進度只保存在本地瀏覽器。後續版本計劃添加雲同步功能。

### Q4: 發音不清楚怎麼辦？
**A:**
- 多點擊「聆聽發音」按鈕
- 嘗試不同瀏覽器
- 調整系統音量

### Q5: 如何清除進度重新開始？
**A:**
```
Settings → Privacy → Clear browsing data
→ 勾選 "Cookies and other site data"
→ Clear
```
