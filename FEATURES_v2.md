# 博幼 800 單：單字爭霸戰 Pro - 完整升級版

## 🎮 遊戲概述

這是一款針對國中生設計的英文單字學習遊戲，包含 40 課程、800 個精選單字，提供三種遊戲模式和多層次難度設置。

---

## ✨ 新增功能列表

### 1. 🎯 難度系統
- **簡單模式**：時間充裕(150ms)，附加 2 生命值，分數 1x 倍數
- **普通模式**：標準難度(100ms)，分數 1.5x 倍數 ⭐ 推薦
- **困難模式**：挑戰難度(60ms)，分數 2.5x 倍數

每個課程都可獨立選擇難度，允許不同程度的玩家挑戰。

### 2. 🏆 成就系統
解鎖 7 個成就徽章：
- **初試啼聲** 🎮 - 完成第一場遊戲
- **十連勝** 🔥 - 連續完成 10 場遊戲
- **完美無缺** ⭐ - 滿血通關一個課程
- **飛快反應** ⚡ - 在 20 秒內答對 5 題
- **全能戰士** 🏆 - 完成所有三種模式
- **集卡達人** ⭐⭐⭐ - 獲得 50 個星星
- **學霸** 🧠 - 獲得 200 個星星

### 3. 📊 進度統計儀表板
實時追蹤：
- 總分數累計
- 星星獲取進度 (當前/360)
- 遊戲次數統計
- 勝率百分比
- 各課程星級進度視圖
- 遊戲歷史記錄

### 4. 🎨 主題系統
- **深色模式** 🌙 - 護眼設計，適合長時間遊玩
- **淺色模式** ☀️ - 明亮界面，日光環境友善
- 一鍵切換，自動保存偏好設置

### 5. 🔊 音效系統
- ✓ 正確答案提示音
- ✓ 錯誤答案提示音
- ✓ 升級成功提示音
- 可在設置中啟用/禁用

### 6. 📈 詳細統計分析
支持計算：
- 平均分數
- 總遊戲局數
- 贏率統計
- 最常玩的模式推薦

### 7. 💾 本地持久化存儲
所有進度通過 LocalStorage 自動保存：
- 遊戲統計數據
- 成就解鎖記錄
- 課程星級進度
- 用戶偏好設置（主題、音效、難度）

### 8. 🎮 增強的遊戲畫面
- 改進的 UI/UX 設計
- 流暢的動畫過渡
- 暫停菜單選項
- 實時計分顯示
- 組合計數器 (Combo)

### 9. 🎓 三種遊戲模式

#### 認讀挑戰 (Recognition)
- 看到中文，選擇正確的英文單字
- UFO 動畫效果
- 防禦塔角色設定

#### 字母重組 (Scramble)
- 將打散的字母重新排列
- 互動式字母磚塊
- 即時驗證答案

#### 拼字測驗 (Spelling)
- 聽到發音後拼出英文
- 語音播放功能
- 鍵盤輸入支持

## 🚀 使用指南

### 遊戲流程
1. **首頁** → 選擇「開始挑戰」
2. **課程選擇** → 選擇 1-40 中的任意課程
3. **難度選擇** → 選擇簡單/普通/困難
4. **模式選擇** → 選擇認讀/字母重組/拼字
5. **遊戲進行** → 回答 20 道題目
6. **結果統計** → 查看得分和需要複習的單字

### 進度查看
- **進度統計** → 查看完整統計數據和課程進度
- **成就系統** → 查看已解鎖和未解鎖的成就
- **主題切換** → 點擊右上角太陽/月亮圖標

## 🎯 計分系統

### 基礎計分
```
基礎分 = 10 + 時間獎勵 + 連擊獎勵
時間獎勵 = 剩餘時間 / 10
連擊獎勵 = 當前連擊數 × 5
```

### 難度倍數
- 簡單：1x（基礎分）
- 普通：1.5x（推薦挑戰）
- 困難：2.5x（高難度挑戰）

### 星級評定
```
剩餘生命值 3 顆 → ⭐⭐⭐ (3 星)
剩餘生命值 2 顆 → ⭐⭐ (2 星)
剩餘生命值 1 顆 → ⭐ (1 星)
生命值用盡 → 0 星
```
